## Tech Stack

- Next.js 15.0.0
- TypeScript
- Tailwind CSS
- JSONPlaceholder API

### Prerequisites

- Node.js 18.x or later
- npm or yarn

### Installation

```bash
git clone https://github.com/payton-ggg/ubiquitous-tribble
npm ci
```

I recommend you to use 'ci' to create workspace exactly as I do.

### Lunch your app

```bash
npm run dev
```
